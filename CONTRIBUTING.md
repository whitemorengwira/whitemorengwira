# Contributing

This repository is maintained as the public GitHub profile for Whitemore Ngwira / N.White Systems.

It is not an open-source project and is not intended for public feature contributions.

Helpful reports are welcome when they improve public review quality:

- Broken public links
- Typos or unclear public documentation
- Accessibility issues in public-facing materials
- Security concerns, handled through [SECURITY.md](SECURITY.md)

Do not submit confidential client data, credentials, private dashboards, internal records, or proprietary implementation details.
