# Code of Conduct

This repository is a professional portfolio surface. Keep all interaction respectful, specific, and relevant to public review quality.

Unacceptable behavior includes harassment, discriminatory language, spam, credential sharing, attempts to expose private data, or pressure to disclose confidential implementation details.

Reports can be sent to [hello@nwhite.systems](mailto:hello@nwhite.systems). Security-sensitive matters should follow [SECURITY.md](SECURITY.md).
